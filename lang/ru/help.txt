==== управление ====
Вы можете управлять картой с помощью мыши или клавиатуры.

== управление мышью/прикосновением ==
  * **перемещение** перемещайтесь по карте, перетаскивая её с нажатой левой кнопкой мыши
  * **обзорная карта** используйте кнопку + в правом нижнем углу карты для отображения обзорной карты
  * **масштабирование** используйте кнопки + и - в левом верхнем углу карты для отображения карты более крупного или мелкого масштаба
  * **переключение тем или карт** щёлкните иконку ≡ справа от карты для просмотра и выбора доступных карт и тем
  * **получение информации** карта может содержать элементы с дополнительной информацией, показываемой во всплывающем окне при щелчке по элементу
  * **полноэкранное отображение** используйте кнопку ✈ для отображения карты на полном экране и кнопку ✕ для возврата к страничному отображению.

== управление с клавиатуры ==
Управление с клавиатуры становится доступным после перехода в область карты клавишей Tab (карта при этом отображается с рамкой фокуса ввода).
  * **перемещение** используйте клавиши со стрелками для перемещения по карте
  * **масштабирование** используйте клавиши + и - для отображения карты более крупного или мелкого масштаба
  * **получение информации** карта может содержать элементы с дополнительной информацией; по нажатию клавиши i появляется курсор, который можно перемещать между элементами клавишами со стрелками; по нажатию Enter информация о выбранном элементе показывается во всплывающем окне; по нажатию клавиш i или Esc происходит возврат в обычный режим
